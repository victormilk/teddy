<task type="auto" role="installer-dev" wave="1">
  <name>Task 2: Add validateConfigDir function</name>
  <files>bin/install.js</files>
  <action>
    Add a validateConfigDir(dirPath) function that:
    1. Rejects paths containing ".." segments (path traversal)
    2. Rejects empty or whitespace-only paths
    3. Resolves the path to absolute and checks it starts with os.homedir() OR process.cwd()
    4. Returns the resolved absolute path on success
    5. Throws an error with descriptive message on failure

    Implementation:
    ```js
    function validateConfigDir(dirPath) {
      if (!dirPath || !dirPath.trim()) {
        throw new Error("Config directory path cannot be empty");
      }
      const resolved = path.resolve(dirPath);
      if (resolved !== path.normalize(resolved) || dirPath.includes("..")) {
        throw new Error(`Invalid config directory: path traversal detected in "${dirPath}"`);
      }
      const home = os.homedir();
      const cwd = process.cwd();
      if (!resolved.startsWith(home) && !resolved.startsWith(cwd)) {
        throw new Error(`Invalid config directory: "${resolved}" must be within home (${home}) or project (${cwd})`);
      }
      return resolved;
    }
    ```

    Include validateConfigDir in module.exports.
  </action>
  <verify>
    node -e "const {validateConfigDir} = require('./bin/install.js'); try { validateConfigDir('../../../etc'); } catch(e) { console.log('BLOCKED:', e.message); }"
    should print "BLOCKED: ..." with traversal error.
  </verify>
  <done>AC-1: validation function exists</done>
</task>
